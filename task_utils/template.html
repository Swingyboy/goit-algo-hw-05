Conclusion

Based on the results, the following conclusions can be made:

  • Algorithm Booyer-Moore Search is the fastest for both texts.
  • Algorithm Rabin-Karp Search is the slowest for both texts.
  • Algorithm Knuth-Morris-Pratt Search is close enough to Booyer-Moore Search for small and big pattern length.
  • All algorithms are quite good for small pattern length.
  • The results are relevant both for the case when the pattern is present in the text and for the case when the pattern is absent
